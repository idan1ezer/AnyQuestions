
# AnyQuestions :man_student: :woman_student:

:question::question: The Class Assistant App is a mobile application built using Flutter. It helps record and organize student questions and professor answers, provides text-based presentation, and offers statistical analysis for professors. :question::question:

## Requirements
To launch the app, you need the following:
- Flutter SDK installed
- IDE: Android Studio / Visual Studio Code / IntelliJ IDEA Community Edition / Emacs (choose one)
- Emulator or physical device for testing


## Getting Started
Follow these steps to run the app locally:
1. Clone the repository:\
  ```git clone https://github.com/idan1ezer/AnyQuestions.git```
2. Open the project in your preferred IDE (e.g., Android Studio, Visual Studio Code).
3. Ensure Flutter dependencies are up to date by running the following command in the terminal:\
  ```flutter pub get```
4. Launch the emulator or connect your physical device.
5. Build and run the app using the following command:\
  ```flutter run```


## FAQ

**Q: How can I contribute to the development of this app?**\
A: We welcome contributions! Please follow the guidelines outlined in CONTRIBUTING.md to get started.

**Q: What is the purpose of the Class Assistant software?**\
A: The Class Assistant software is designed to help students and professors in a classroom setting. It records student questions and professor answers, provides text-based presentation, and offers statistical analysis for professors.

**Q: How does the Class Assistant software benefit students?**\
A: The software aims to benefit students in several ways:
   - Avoiding repeated questions to save time and enhance efficiency.
   - Helping students regain focus if they get distracted during lectures.
   - Identifying difficult topics that require additional attention.
   - Increasing student involvement and engagement in class.

**Q: What are the benefits for professors?**\
A: Professors can benefit from the Class Assistant software through the following:
   - Access to statistical analysis that provides insights into student questions and answers.
   - Tools for identifying patterns and common issues in different courses, groups, or classes.
   - Aids in creating a more interactive and inclusive learning environment.
   - Assistance for students with hearing difficulties through text-based presentation.

**Q: What technology is used to develop the Class Assistant mobile application?**\
A: The Class Assistant mobile application is developed using the Flutter framework, which allows for cross-platform development for both Android and iOS.

**Q: Is the Class Assistant software open source?**\
A: Yes, the Class Assistant software is open source. You can find the project repository on GitHub under an open source license.


## How to Use
1. Launch the app on your mobile device or emulator.
2. Sign in with your credentials or create a new account.
3. Navigate through the courses, groups, and classes to view the questions and answers.
4. Use the search functionality to find specific topics or questions.
5. Access the statistics section to analyze question patterns and overall engagement.
